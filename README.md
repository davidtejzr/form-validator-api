## Project setup

```bash
$ npm install
```

### Install tools for data providers
```bash
cd data-providers
bash prepare-environment.sh
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test
```
